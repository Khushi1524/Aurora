Aurora is a web platform designed for users to explore, learn, and share experiences related to the Northern Lights.
The project focuses on creating an immersive and visually engaging interface where users can discover information, view experiences, and connect through shared stories about this natural phenomenon.

Features

1. Explore information about the Northern Lights
2. Share and view user experiences
3. Visually engaging and immersive UI
4. Clean and structured layout
5. Responsive design

Tech Stack :

1. HTML
2. CSS
3. JavaScript
4. PHP

Purpose :

This project was created to:

1. Build an experience-based platform
2. Improve UI/UX design skills
3. Practice full-stack development
4. Create a unique and meaningful web concept

Inspiration :

Inspired by the beauty and mystery of the Northern Lights, the design focuses on dark themes, glowing elements, and a visually rich experience.

Note :

This project was developed as part of learning and exploring creative web development ideas.
